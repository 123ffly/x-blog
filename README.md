## 个人博客
  这是一个开源的个人`blog`项目.主要目的是玩一玩`vue`。大家可以在本项目中不仅仅可以学习了解以下的`框架/库`，同时还可以了解关于`代码结构组织`,`模块化`,`前端构建`等内容。这个项目才刚开始，我会带着大家完成整个`blog`项目的开发工作。

## 技术栈

前端
* es6
* vue2
* webpack2
* vue-router2
* axios
* less

后端
* node
* mongoDB


## 起手式

```javascript
  node -v
  v6.9.2

  npm -v
  3.10.9
```

```javascript
  fork本项目

  git clone ...

  cd x-blog

  npm install （建议使用淘宝镜像）
  
  npm run dev
```

## 个人博客
  这是一个开源的个人blog项目.主要目的是玩一玩vue。大家可以在本项目中不仅仅可以学习了解以下的框架/库，同时还可以了解关于代码结构组织,模块化,前端构建等内容。

## 技术栈

前端
* es6
* vue2
* webpack2
* vue-router2
* axios
* less

后端
* node
* mongoDB


起手式:

```javascript
  node -v
  v6.9.2

  npm -v
  3.10.9
```

```javascript
  git clone ...

  cd x-blog

  npm install （建议使用淘宝镜像）
  
  npm run dev
```
